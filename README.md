# CV Builder
<img width="1896" height="1150" alt="image" src="https://github.com/user-attachments/assets/225edc56-3b9d-4ff1-b748-12e0ea9f09a0" />

A modern, full-stack CV/Resume builder application that allows users to create, customize, and export professional resumes with ease.

## Features

- **Dynamic CV Generation**: Create personalized resumes with customizable sections
- **Multiple Variants**: Support for different CV variants (Software Engineer, Hardware/FPGA, AI/ML, Web Development, etc.)
- **Real-time Editing**: Live preview of your CV as you make changes
- **Export to PDF**: Generate professional PDF exports of your resume
- **Data Persistence**: Local database storage for your CV data
- **Responsive Design**: Clean, modern UI built with React

## Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Backend**: Express.js, Node.js
- **Styling**: CSS
- **Build Tool**: Vite
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ahmedfathy0-0/cv-builder.git
cd cv-builder
```

2. Install dependencies:
```bash
npm install
```

3. Start the development servers:
```bash
# Start both frontend and backend
npm run dev:all

# Or start them separately
npm run dev      # Frontend only
npm run server   # Backend only
```

4. Open your browser and navigate to `http://localhost:5173`

## Usage

1. Fill in your personal information in the provided form
2. Add your education, work experience, projects, and skills
3. Choose from different CV variants to highlight specific expertise
4. Preview your CV in real-time
5. Export to PDF when ready
